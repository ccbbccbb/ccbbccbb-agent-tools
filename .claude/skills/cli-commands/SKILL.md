# CLI Tools

## Quick Reference

### Viewing
```bash
tree -L 2                     # directory tree
```

### Git
```bash
gh pr list                    # GitHub PRs
# delta: auto-configured as git diff pager
```

### Processing
```bash
jq '.field'                   # parse JSON
```
